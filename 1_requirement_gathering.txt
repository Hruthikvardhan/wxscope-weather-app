============================================================
WXSCOPE — REQUIREMENT GATHERING
============================================================

PROBLEM STATEMENT
-----------------
Users want to check live weather quickly.
Existing apps are cluttered with ads.
One clean page that shows everything = solution.

WHO IS THE USER?
----------------
- Students
- Travellers
- Working professionals
- Anyone who checks weather daily

WHAT PROBLEM AM I SOLVING?
---------------------------
- Get live weather for any city
- Auto detect current location
- See 5 day forecast in one place
- No page reload needed

WHAT IS THE MVP?
----------------
Search bar + Show current weather
Everything else is extra.

------------------------------------------------------------
TECH STACK DECISIONS
------------------------------------------------------------

Frontend   : HTML5, CSS3, Vanilla JavaScript
API        : OpenWeather API (Free Tier)
Storage    : localStorage (search history)
Fonts      : Google Fonts (Bebas Neue, Orbitron, Share Tech Mono)
Deploy     : Vercel
Repo       : GitHub

------------------------------------------------------------
API ENDPOINTS NEEDED
------------------------------------------------------------

Current Weather by City:
https://api.openweathermap.org/data/2.5/weather?q={city}&units=metric&appid={key}

5 Day Forecast by City:
https://api.openweathermap.org/data/2.5/forecast?q={city}&units=metric&appid={key}

Current Weather by GPS:
https://api.openweathermap.org/data/2.5/weather?lat={lat}&lon={lon}&units=metric&appid={key}

5 Day Forecast by GPS:
https://api.openweathermap.org/data/2.5/forecast?lat={lat}&lon={lon}&units=metric&appid={key}
